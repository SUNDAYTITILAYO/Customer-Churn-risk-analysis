# Customer Churn & Risk Analysis Dashboard

## Project Overview
This project analyses customer churn behaviour using Python and Tableau to identify high-risk customer segments and support data-driven retention and pricing strategies. A predictive model was developed to estimate churn probability, and insights were visualised through an interactive dashboard.

## Skills Demonstrated
- Data Cleaning & Preprocessing
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Predictive Modelling (Logistic Regression)
- Data Visualisation (Tableau)
- Business Insight Communication

## Tools Used
- Python
- Pandas
- Scikit-learn
- Tableau Public
- Excel

## Project Workflow
1. Cleaned and prepared customer data in Python
2. Handled missing values and corrected data types
3. Engineered features such as tenure groups and value segments
4. Built a logistic regression model to generate churn risk scores
5. Created an interactive Tableau dashboard to visualise churn trends and customer risk segments

## Key Insights
- Overall churn rate was **26.5%**
- Customers on **month-to-month contracts** had the highest churn rate at **42.7%**
- Customers on **two-year contracts** had the lowest churn rate at **2.8%**
- The dashboard identified a meaningful group of **high-risk, high-value customers**, supporting targeted retention strategies

## Dashboard Features
- Overall churn KPI
- Churn rate by contract type
- Risk vs value segmentation heatmap
- Interactive filters for contract type, value segment, and risk segment
